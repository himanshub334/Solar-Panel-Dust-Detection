# Solar Panel Dust Detection using Deep Learning

This project uses MobileNetV2 transfer learning to classify solar panels as Clean or Dusty using image data.

## Tech Stack
- Python
- TensorFlow
- OpenCV
- CNN (MobileNetV2)

## Accuracy
Achieved 90%+ validation accuracy.

## Future Improvements
- Real-time camera integration
- Dust percentage estimation
- IoT-based cleaning alert system
